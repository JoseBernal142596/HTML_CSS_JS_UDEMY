Color por código hsl

Este es un ejemplo con color con código hsla.
Este funciona a través de una rueda de colores y los colores se escogen por los ángulos donde estén ubicados, es decir, si queremos el color verde aguamarina escogemos aproximadamente el 210.
La estructura es: ejemplo: hsla(348, 83% (escala de grises d0% es gris y 100% es el color completo),40(iluminación (0% es negro y 100% blanco)))

Color con código hexadecimal

Color con código hexadecimal Este es un ejemplo de color de fondo como color de letra con código hexadecimal. Un ejemplo de código hexadecimal es: #2E4053. Pude tener como máximo 16 caracteres donde 0 es el valor más bajo y f el más alto (000000 es el color negro y ffffff es el color blanco).
