Estilos CSS con ID

Este es un ejemplo de un div con un id el cuál va a ser editado en CSS utilizando #.

Y este es un div con el Selector Class. Para editar una clase en CSS utilizamos: , .clase (Un punto y el nombre de la clase).
La diferencia de Class con ID es que una clase la podemos usar para varias Etiquetas y el ID es para un elemento único, es decir, que no se repita.
