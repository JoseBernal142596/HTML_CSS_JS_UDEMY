Elementos Inline y block (en línea y en bloque)


Elementos Inline (en linea)-(span)

Los elementos en linea son los elementos que están en un mismo parrafo o renglón y pueden ser modificados por aparte sin necesidad de un nuevo bloque y para eso utilizamos la etiqueta (span).



Elementos block (en bloque)-(div)

Este es el título del div que es la etiqueta para crear los bloques

Este es un parrafo dentro de la etiqueta (div) y este bloque está conformado por este parrafo y el título.

Este es un parrafo fuera del div.
