Formularios en HTML

Para crear un formulario en HTML, utilizamos a etiqueta < form> < /form>.
Para que el usuario nos dé información como su nombre, apellido, etc usamos la etiqueta input.







Para validar un formulario, es decir que el formulario no se envie hasta que los campos del mismo estén completos utilizamos el atributo de required








Método GET y POST HTTP

Por lo general en las URL se puede ver la información del formulario. Por ejemplo formulariosenhtml.html?nombre=Juan&apellido=Gutiérrez lo que está en azul se le conoce como Query string que vamos a enviar hacia el servidor.

Sin embargo, si tenemos un campo sensible o privado como la contraseña y no queremos que se vea en el Query string entonces debemos cambiar el tipo de método para envíar la información al servidor.

Por defecto, el tipo de método que se usa para enviar la información al servidor es el método GET, pero podemos utilizar el método POST para enviar este formulario hacia el servidor y la información en lugar de que se agregue en la URL, se va a envíar dentro del cuerpo del mensaje que se envía hacia el servidor.

Para utilizar el metodo post utilizamos el atributo method="post" < form method="post" >
























Para agregar valores numéricos con decimales, utilizamos step="any"
