Se hace con la etiqueta pre y sirve cuando queremos mostra en la página un párrafo tal cual como lo estamos escribiendo


Para escribir en negrita podemos utilizar las etiquetas b y strong, sin embargo, strong se usa cuando queremos decirf que ese texto o palabra es más importante


Para poner un texto en cursiva utilizamos la etiquera i o también para enfatizar usamos la etiqueta em


Para escribir un texto más pequeño utilizamos la etiqueta small


Para agregar un texto resaltado utilizamos la etiqueta mark


Para escribir un texto eliminado utilizamos la etiqueta del


Para indicar que se ha insertado un texto utilizamos la etiqueta ins


Para indicar un subindice utilizamos la etiqueta sub


Para indicar un Superindice, utilizamos la etiqueta sup


Si queremos escribir algo Con letras parecidas a las de un código utilizamos la etiqueta code.


También podemos buscar cómo escribir caracteres como porcentaje entre otros usando https://dev.w3.org/html5/html-author/charref.
